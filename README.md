# aspnet-mvc-labs
A collection of ASP.NET MVC labs built during my Higher Diploma in Computer Science at TU Dublin

## TextMessage — SMS Sender App

An ASP.NET MVC app that allows a user to compose and send a text message with validation.

**Features**
- Form with mobile number (10 digits) and message content (max 140 chars)
- Model validation attributes enforcing input rules
- Confirmation page showing sent message details with link to send another

**Concepts Demonstrated**
- MVC architecture
- Data annotation validation attributes
- Scaffolded Edit view
- Controller redirects between views

**Tech:** C#, ASP.NET MVC, Razor Views

## Groceryy — Online Grocery Shop

An ASP.NET MVC app that displays a list of grocery products which can be added to a shopping cart and purchased.

**Features**
- Product listing page with names and prices
- Add items to a shopping cart
- Cart summary showing items and total amount
- Checkout page where users can input payment details
- Payment confirmation flow (in progress)

**Concepts Demonstrated**
- MVC architecture
- Model relationships (products, cart, cart items)
- Session/cart management
- Form handling and input validation
- Controller navigation between views

**Tech:** C#, ASP.NET MVC, Razor Views
